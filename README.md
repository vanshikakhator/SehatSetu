# GramCare

Full-stack healthcare ecosystem for rural India.

## Project Structure

- `frontend/`: React + Vite application
  - `src/components/`: Reusable UI components and modals
  - `src/pages/`: Main application pages and dashboards
  - `src/constants/`: Mock data and theme constants
- `backend/`: Node.js + Express server

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```
